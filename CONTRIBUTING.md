# Contribuir al proyecto

Gracias por tu interés en contribuir a este proyecto.

## Cómo contribuir

1. Haz un fork del repositorio
2. Crea una rama para tu cambio:
   git checkout -b feature/nombre-de-tu-cambio
3. Realiza tus cambios
4. Haz commit:
   git commit -m "Descripción clara del cambio"
5. Sube tu rama:
   git push origin feature/nombre-de-tu-cambio
6. Abre un Pull Request

## Reglas básicas

- Mantén el código limpio y organizado
- Usa nombres descriptivos
- No subas código innecesario
- Sigue la estructura del proyecto

## Issues

Antes de crear un issue:
- Revisa si ya existe uno similar
- Explica claramente el problema o la propuesta

## Notas

Este proyecto es mantenido actualmente por una sola persona, por lo que las decisiones finales sobre contribuciones recaen en el mantenedor.

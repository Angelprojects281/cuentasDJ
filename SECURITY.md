# Política de Seguridad

## Versiones soportadas

Actualmente, este proyecto se encuentra en desarrollo activo, por lo que solo se da soporte a la versión más reciente disponible en el repositorio.

| Versión | Soporte          |
| ------- | ---------------- |
| Última  | :white_check_mark: |
| Antiguas| :x:              |

## Reportar una vulnerabilidad

Si encuentras una vulnerabilidad de seguridad, por favor repórtala de manera responsable.

**No abras un issue público para este tipo de reportes.**

En su lugar, puedes:

- Enviar un correo a: angel15laverde16@gmail.com 
- O contactar directamente al mantenedor del proyecto  

## Qué puedes esperar

- Se intentará responder en un plazo de 48 a 72 horas  
- Se evaluará la validez del reporte  
- Si la vulnerabilidad es confirmada, se trabajará en una solución lo antes posible  
- En algunos casos, puede que no se proporcione información detallada hasta que el problema sea resuelto  

## Alcance

Esta política aplica a todo el proyecto y a su versión más reciente.

## Notas

Este proyecto es mantenido por una sola persona, por lo que los tiempos de respuesta pueden variar.
